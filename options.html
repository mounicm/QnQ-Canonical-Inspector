QnQ Canonical Inspector

Full Credit to original extension author Tobias Redmann

There are three states of the canonical url:

1.) no (Grey) icon: The current website has no canonical url specified.

2.) Green icon: The website urls matches the canonical url.

3.) Red icon: The canonical url for this site is another url. Click on the icon to go to the specified canonical url.
